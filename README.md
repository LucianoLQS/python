# Proyecto de Python - UNAC

Este repositorio contiene ejercicios de Python realizados para la clase en UNAC. Cada archivo implementa un pequeño programa interactivo para practicar condicionales, entradas de usuario y cálculos básicos.

## Archivos incluidos

- `admin.py`
  - Verifica si el usuario y la contraseña ingresados coinciden con las credenciales de administrador.
  - Si el usuario es `admin` y la contraseña es `1234`, muestra un mensaje de bienvenida.
  - Si no, muestra un mensaje indicando que los datos son incorrectos.

- `colegio.py`
  - Pide un número al usuario y calcula su valor multiplicado por 2.
  - Luego muestra el resultado dentro de un mensaje.

- `nuevo.py`
  - Solicita el nombre, la edad y si el usuario tiene suscripción premium.
  - Determina el nivel de acceso según la edad y el estado de suscripción.
  - Muestra un mensaje personalizado con el acceso completo, limitado o denegado.

- `pytohn.py`
  - Calcula el promedio de tres notas ingresadas por el usuario.
  - Valida el promedio y muestra:
    - "Error" si el promedio es mayor a 20.
    - "Desaprobado" si es menor a 11.
    - "Aprobado" si es mayor a 11.
    - "Normal" si es exactamente 11.

## Notas

- Este proyecto es un buen ejemplo de práctica con entradas de usuario y estructuras condicionales.
- Puedes mejorar estos ejercicios añadiendo manejo de errores y validaciones extra.
